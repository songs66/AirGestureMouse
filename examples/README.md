# Examples

这里保留项目早期迭代脚本，适合学习每一步是如何从摄像头测试逐渐演进到完整空气鼠标的。

| 文件 | 作用 |
| --- | --- |
| `prototypes/01_camera_test.py` | 摄像头读取与窗口显示测试 |
| `prototypes/02_hand_detect_test.py` | MediaPipe 手部关键点检测测试 |
| `prototypes/03_mouse_move_test.py` | 食指映射鼠标移动测试 |
| `prototypes/04_left_click_test.py` | 左键点击手势测试 |
| `prototypes/05_full_gesture_mouse.py` | 完整功能早期版本 |
| `prototypes/06_final_gesture_mouse.py` | 重构前的最终单文件版本 |

正式运行请优先使用根目录 README 中的安装方式和 `air-gesture-mouse` 命令。
